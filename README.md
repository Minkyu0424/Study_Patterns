## Study_Patterns 디자인 패턴에 대해 공부하고 기록합니다.

### 다음 요소들을 고려하며 패턴들을 공부하자.

- 패턴의 이름과 패턴의 역할을 매칭
- 패턴을 사용하는 경우 문제 상황 인지
- 문제에 대해 패턴을 통해 해결하는 방법 인지
- 패턴의 결과와 다른 패턴과 비교해 장단점을 인지
- 패턴에 대한 예제코드 이해하고 따라 작성해보기
